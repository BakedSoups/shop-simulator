Shop Simulator Art Spec

Goal
- Replace the current placeholder PNGs with a coherent pixel-art set that matches the current game layout and interactions.
- Current world is a compact PlateUp-style room:
  - modular carrot farm on the left
  - juicer and supply area in the middle
  - shelf/shop area on the right
  - storefront wall + door + street outside

Base Sizes
- World tile size in game logic: 40x40
- Source art can be 16x16, 32x32, or 48x48, but should be exported/cropped per usable object
- Recommended final replacements:
  - ground/walls/furniture/machines: one image per tile footprint
  - hand-held items: small centered sprite with transparent padding
  - character: directional or animated frames if available
